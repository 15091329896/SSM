分层学习的基本思路
    全面，短小
    全面分小阶段
    
    有序推进
    
    核心原则
    分阶段全面学习技术
    
    知识点的挑选和划分
    对于同一个技术点，不同水平的程序员掌握的程度是不应该是相同的。
    知识点的区别不应该是学过和没学过，而应该是学习的程度是怎样的
    在一套只是当中找出和自己目前的职位级别向匹配的知识点进行学习

基本做法是
进行知识点的挑选和划分，将知识点划分成不同的层级，然后一层一层学习

低级水平最迫切需要的知识点是什么
中级水平迫切需要的知识点是什么
高级水平迫切需要的知识点是什么

编写教材：
划分出初级水平的程序员学习的资料库
    初级划分为3个阶段
适合中级程序员学习的资料库
    中级划分为3个阶段
适合高级程序员学习的资料库
    高级学习划分为3个阶段
    

    选择有效的方式学习
    分阶段全面学习指定大纲
    按照大纲搜集对应的资料进行学习：资料的优先级是视频，音频，文字
